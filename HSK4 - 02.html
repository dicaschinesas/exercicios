Exercícios de Gramática Chinesa: HSK 4 - GRAMÁTICA 02 - 死了

Instruções: 1-5: Responda às perguntas. 6-10: Reordene as frases.

1. Qual das seguintes frases usa 死了 correctamente para intensificar um adjectivo?

2. O que significa 死了 na frase 今天热死了?

3. Qual frase significa "Estou a morrer de fome"?

4. Qual é a estrutura correcta para usar 死了?

5. Qual frase usa 死了 para intensificar um adjectivo de forma positiva?

6. Reordene as palavras para formar uma frase correcta:

饿 / 我 / 死了 / 。

7. Reordene as palavras para formar uma frase correcta:

热 / 今天 / 死了 / ！

8. Reordene as palavras para formar uma frase correcta:

烦 / 这 / 部 / 电影 / 死了 / ！

9. Reordene as palavras para formar uma frase correcta:

高兴 / 我 / 死了 / ！

10. Reordene as palavras para formar uma frase correcta:

累 / 我 / 死了 / 。
