Exercícios de Gramática Chinesa: HSK 4 - GRAMÁTICA 06 - 从来

Instruções: 1-5: Responda às perguntas. 6-10: Reordene as frases.

1. Qual das seguintes frases usa 从来 correctamente para significar "nunca"?

2. O que significa 从来 na frase 你从来不给我打电话?

3. Qual frase significa "Eu nunca fui ao Brasil"?

4. Qual é a estrutura correcta para 从来 + 不?

5. Qual frase usa 从来 + 没 para expressar algo que nunca foi feito?

6. Reordene as palavras para formar uma frase correcta:

从来 / 他 / 不 / 喝酒 / 。

7. Reordene as palavras para formar uma frase correcta:

我从来 / 你 / 不 / 给 / 打电话。

8. Reordene as palavras para formar uma frase correcta:

过 / 从来 / 我 / 没 / 去 / 法国。

9. Reordene as palavras para formar uma frase correcta:

从来 / 我 / 没 / 尝 / 过 / 这道菜 / 。

10. Reordene as palavras para formar uma frase correcta:

从来 / 她 / 肉 / 吃 / 不 / 。
