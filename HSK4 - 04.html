Exercícios de Gramática Chinesa: HSK 4 - GRAMÁTICA 04 - 不仅...而且...

Instruções: 1-5: Responda às perguntas. 6-10: Reordene as frases.

1. Qual das seguintes frases usa 不仅...而且... correctamente?

2. O que significa 不仅 na frase 桂林不仅城市好看，人也很热情?

3. Qual frase significa "Este prato não só é delicioso como também é bem nutritivo"?

4. Qual é a estrutura correta para 不仅...而且...?

5. Qual frase usa 不仅...而且... para expressar duas características positivas?

6. Reordene as palavras para formar uma frase correcta:

不仅 / 这件 / 衣服 / 好看 / ， / 而且 / 便宜 / 。

7. Reordene as palavras para formar uma frase correcta:

不仅 / 桂林 / 城市 / 好看 / ， / 人 / 也 / 很 / 热情 / 。

8. Reordene as palavras para formar uma frase correcta:

不仅 / 这道菜 / 好吃 / ， / 还 / 很 / 有 / 营养 / 。

9. Reordene as palavras para formar uma frase correcta:

不仅 / 我 / 喜欢 / 旅行 / ， / 我的 / 爸爸妈妈 / 也 / 喜欢 / 旅行 / 。

10. Reordene as palavras para formar uma frase correcta:
