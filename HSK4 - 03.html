Exercícios de Gramática Chinesa: HSK 4 - GRAMÁTICA 03 - 要是..., 就...

Instruções: 1-5: Responda às perguntas. 6-10: Reordene as frases.

1. Qual das seguintes frases usa 要是..., 就... correctamente?

2. O que significa 要是 na frase 要是你的飞机不晚点，我们就可以去那家餐厅吃饭了?

3. Qual frase significa "Se acabares cedo, liga-me"?

4. Qual é a estrutura correcta para 要是..., 就...?

5. Qual frase usa 要是..., 就... para expressar uma condição?

6. Reordene as palavras para formar uma frase correcta:

要是 / 你 / 有 / 时间 / ， / 就 / 帮 / 我 / 打扫 / 房间 / 。

7. Reordene as palavras para formar uma frase correcta:

9. Reordene as palavras para formar uma frase correcta:
